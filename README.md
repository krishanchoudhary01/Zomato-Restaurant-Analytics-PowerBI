# 🍽️ Zomato Restaurant Analytics Dashboard (Power BI)

## 📌 Project Overview

The **Zomato Restaurant Analytics Dashboard** is an interactive Power BI project designed to analyze restaurant performance, customer behavior, online delivery trends, pricing, ratings, and business insights. This dashboard transforms raw restaurant data into meaningful visualizations that help businesses make data-driven decisions.

---

## 🎯 Project Objectives

- Analyze restaurant distribution across different cities.
- Identify top-performing restaurants based on ratings and customer votes.
- Study customer engagement through online delivery and table booking.
- Compare restaurant pricing and average ratings.
- Discover the most popular cuisines.
- Generate business insights and executive recommendations.

# 📊 Key Insights

- Restaurants with online delivery receive better customer engagement.
- North Indian cuisine dominates the dataset.
- Mid-range restaurants attract the highest number of customers.
- Only a small percentage of restaurants provide table booking.
- Customer ratings remain consistent among top-performing restaurants.
- Major cities contribute the highest number of restaurants and customer votes.
